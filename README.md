# React Scheduler Demo

Интерактивный учебный проект по конкурентному рендерингу в React.
Основан на главе 7 книги **«Конкурентный React»** (стр. 223-259).

## Разделы

| # | Раздел | Что демонстрирует |
|---|--------|-------------------|
| 1 | **Медленный чат** | Проблема: синхронные обновления блокируют UI |
| 2 | **Быстрый чат** | Решение через `useTransition` + A/B переключатель |
| 3 | **Планировщик React** | Визуализация `ensureRootIsScheduled`, lanes, очереди задач |
| 4 | **Event Loop** | Интерактивная диаграмма браузерного event loop |
| 5 | **useDeferredValue** | Поиск по 20 000 элементов, прерываемый рендер |
| 6 | **Разрыв (Tearing)** | Конкурентный разрыв состояния и `useSyncExternalStore` |
| 7 | **Проверь себя** | 5 вопросов из книги с разбором ответов |

## Запуск

```bash
npm install
npm run dev
```

Откроется на `http://localhost:5173`.

## Стек

- React 19 + TypeScript + Vite
- Framer Motion — анимации
- CSS — тёмная тема, без UI-библиотек

## Структура

```
src/
├── App.tsx                      — роутинг между разделами
├── main.tsx                     — точка входа
├── components/                  — переиспользуемые компоненты (чат, FPS-метр)
├── sections/                    — корневые компоненты разделов
├── visualizer/                  — внутренности визуализатора планировщика
└── styles/global.css
```
